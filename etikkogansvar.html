Etikk og ansvar.

KNE driver virksomheten i tråd med høye krav til etikk, seriøsitet og åpenhet. Våre etiske retningslinjer legger føringer for hvordan vi forventer at både ansatte, leverandører og samarbeidspartnere opptrer i tråd med våre verdier og gjeldende lover og regler.

Varslingsrutiner

Trygg rapportering.

KNE har en tydelig og uavhengig varslingsordning der ansatte, leverandører, samarbeidspartnere og andre kan melde fra om kritikkverdige forhold. Varsling kan gjøres anonymt, og det er ikke tillatt med represalier mot personer som varsler i god tro.

Hva kan varsles?

Varsling gjelder forhold som er kritikkverdige, og som kan ha betydning for arbeidsmiljø, sikkerhet, etikk eller lovlighet i virksomheten. Eksempler er brudd på lover og forskrifter, fare for liv og helse, diskriminering eller trakassering, økonomiske misligheter, korrupsjon, alvorlige brudd på HMS-rutiner, eller annen uetisk adferd.

Uenighet om faglige vurderinger eller ordinære personalsaker regnes normalt ikke som varsling.

Hvordan behandler vi varsler?

Alle varsler behandles konfidensielt, objektivt og så raskt som mulig, i tråd med våre interne rutiner og arbeidsmiljølovens krav. Varslet vurderes først for å avklare om det faller inn under ordningen. Deretter gjennomføres nødvendige undersøkelser, som kan inkludere samtaler, dokumentinnhenting og faktasjekk.

Varsleren skal ikke oppleve negative følger, og saken håndteres med respekt for både varsler og den eller de varselet gjelder.

Hvem mottar varselet?

Varsler kan sendes til nærmeste leder, HR-funksjon eller daglig leder, avhengig av hva varselet gjelder. Dersom saken gjør det vanskelig å varsle i linjen, kan du sende varsel direkte via vårt varslings­skjema nederst på denne siden.

Skjemaet går til en dedikert varslings­mottaker som sikrer at saken håndteres konfidensielt og i tråd med våre rutiner. Mottaker vurderer varselet, igangsetter nødvendige undersøkelser og sørger for videre oppfølging.

Hvor raskt får man tilbakemelding?

Varsler får en bekreftelse på mottak innen få dager etter at varselet er sendt. Innen rimelig tid gis også en overordnet orientering om hvordan saken vil følges opp, og hvem som behandler den.

Varsleren får informasjon når saken er avsluttet, så langt det er mulig uten å gå på bekostning av taushetsplikt eller andre involvertes personvern.
